# Arduino-Infrared-Obstacle-Avoidance-Sensor-with-Buzzer-Alarm- 

Learn how to detect obstacles with an Arduino using an Infrared (IR) Obstacle Avoidance Sensor and activate a buzzer alarm when an object is detected.

Please Subscribe To The Channel For More Videos.

Wiring:

S = Signal → Connect to Arduino pin 3
V = VCC (Voltage) → Connect to 5V
G = Ground → Connect to GND

Buzzer Wiring:

+ (Positive) → Connect to Arduino pin 13
– (Negative) → Connect to GND

Code behavior:

• IR sensor is connected to pin 3
• Buzzer is connected to pin 13
• When an object is detected, the sensor output goes LOW
• The buzzer beeps rapidly (80ms ON / 80ms OFF)
• When no object is detected, the buzzer remains OFF

Perfect for beginners learning Arduino programming, infrared sensors, obstacle detection, digital inputs, and buzzer projects.
